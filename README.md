Slot Booking Website

A simple MERN stack application that allows users to book available slots and admins to manage slot availability.

Tech Stack

Frontend: React, Tailwind CSS, Vite

Backend: Node.js, Express.js

Database: MongoDB

Auth: JWT

Features

User Signup & Login

Admin & User roles

Admin can create and delete slots

Users can view and book available slots

Users can cancel bookings

Admin can view all bookings

Responsive UI
